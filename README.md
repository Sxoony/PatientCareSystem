# PatientCareSystem


C# Healthcare system that can add, remove, search for patients, add them to a file, etc.


## Description
This is an application that gives users the ability to add patients, remove them, and display patients.


## Features
- Display patients
- Add patients
- Remove patients
- Add patients to a textfile
- Search for patients


## Technologies
- Language / Framework: C#


## How to run (local)
1. Clone: `git clone https://github.com/Sxoony/PatientCareSystem.git`
2. Navigate: `cd REPO-NAME`
3. Setup:
- Open in Visual Studio (C#)
4. Run
